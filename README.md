# YTDLP CLI
yt-dlp is an amazing command-line a/v downloader with a ton of features,
in fact i recommend checking out their repo https://github.com/yt-dlp/yt-dlp

#Includes:
1. Ready to download folder with .bat file for an easy to use ytdlp experience.
2. Options to download Video and audio or Audio only using ffmpeg(also included within RUN file)

#Instructions:
1. Download full repo using CODE > Download Zip
2. Unzip .zip file
3. Enter "src" folder
4. run RUN.bat
